Shared ESLint configuration for my projects.

## Installation

```sh
npm install --save-dev @bricked/eslint-config
yarn add --dev @bricked/eslint-config
pnpm add --dev @bricked/eslint-config
```

## Usage

Add the ESLint config to your `package.json`:

```json
{
  "eslintConfig": {
    "extends": "@bricked"
  }
}
```

Or to `.eslintrc`:

```json
{
  "extends": "@bricked"
}
```
